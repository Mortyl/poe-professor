## Weapon-Skill Restriction
In Path of Exile 2, skill gems are tied to specific weapon types. The skills listed in this file can **only** be used when this weapon type is equipped. Do not suggest these skills for builds using a different weapon type, and do not suggest skills from other weapon types for this build.
